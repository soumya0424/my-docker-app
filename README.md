# my-docker-app

A Dockerized Python web application.

## Tech Stack

- Python
- Docker / Dockerfile
- Dev Containers

## Getting Started

### Prerequisites

- [Docker](https://www.docker.com/) installed on your machine

### Run the App

```bash
docker build -t my-docker-app .
docker run -p 5000:5000 my-docker-app
```

## Project Structure

```
my-docker-app/
├── Dockerfile
├── app.py
├── devcontainer.json
└── .gitignore
```
